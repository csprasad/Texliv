# Texliv
Texliv is a demo application for getBasis interview.

## Installation
- Download ZIP
- Extract file
- Open Texliv folder
- Double Tap file named Texliv.xcodeproj (xcode must be installed)
- Select simulator to run
- Then hit run button
- Once app is loaded on simulator you are good to go on testing.
